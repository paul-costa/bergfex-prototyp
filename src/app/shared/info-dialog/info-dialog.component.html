Filter Selection

This side navigation section allows you to select any amount of possible categories to filter by.

For example: If you want to select ski-areas in a certain country and provide the minimum snow-height, select the Country and Minimum Snow Height checkboxes and hit "Filter Selection".

Next, the filter selection is going to minimize and a stepper above the table of ski areas is provided, which guides you through you selected properties.

At any time you can hit the "Reset Selection" button in side navigation or the "Reset" button in the Stepper to restart the process.
