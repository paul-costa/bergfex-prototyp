Bergfex - Skiarea Selector: Prototyp

This prototype simulates the ski-area selector option from the original Bergfex.at website, as well as the rough layout.

This prototype is not yet optimized for mobile view, but only for desktop usage with a horizontal screen size of 1024px or higher.

To access the ski-area selector prototype navigate to "Winter" -> "Ski-Areas" in the header or click here.
